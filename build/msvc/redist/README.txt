=================================================
Visual C++ Redistributable for Visual Studio 2019
=================================================

Download page:
  https://www.visualstudio.com/downloads/
  Scroll down to bottom of page, select "Other Tools and Frameworks" and
  then "Microsoft Visual C++ Redistributable for Visual Studio 2019".

Direct links:
  Win32: https://aka.ms/vs/16/release/vc_redist.x86.exe
  Win64: https://aka.ms/vs/16/release/vc_redist.x64.exe

Various versions of the Visual C++ Redistributable have slightly different file
names. For consistency, we rename them as vcredist32.exe and vcredist64.exe
so that all installers of TSDuck have the same names.
